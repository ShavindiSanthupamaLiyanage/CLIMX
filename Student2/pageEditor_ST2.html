Home Page

The Home Page serves as the central introduction to your website and establishes its visual identity, navigation, and purpose.

Technical Description

HTML Techniques Used

  • Header & Navigation Bar

    This section contains website logo and navigation menu.The main-nav class contains a list of links ul with lithat help users navigate between different pages of the site, such as "Home," "Volunteer," "Tables," "User Profile," "Feedback," "Sitemap," and "Team".

  • Hero Section

    The hero section includes a video background with a muted autoplay feature. The video is embedded using the <video> element with a source tag pointing to the video file. The hero content is displayed over the video with the site title "CLIMX" and the tagline.

  • Description Section

    This section provides a brief overview of the website's purpose and mission.

  • Sub-Content Section

    Contains main 4 sub topics using <sub-content-section> with background images and text content.

  • Footer Section

    The footer contains the site logo, the tagline and social media icons. It also includes sections for editors, editor pages, validation pages, and copyright information.

CSS Techniques

  • Flexbox Layout

    use <display: flex >, <display: inline-block> to create flexible layouts.

  • Grid Layout

    grid layout is applied using <display: grid><grid-template-columns: repeat(auto-fit, minmax(250px, auto))>for footer.

  • Text Effects and Typography

    Links have hover effects such as changing the background color, font weight, and color.

  • Positioning and Z-Index

    <position: absolute,> allowing it to cover the entire hero section behind the content (z-index: -1 ensures it's behind the text).

Accessibility

Challenges and Lessons Learned

Challenges & Solutions

  • Challenge: Ensuring the images load correctly across different div sizes.
  • Solution: Used background-size: cover; and background-position: center; to align the images.
  • Challenge: Ensuring proper alignment of elements across screen sizes.
  • Solution: Used display: flex; and media queries for layout responsiveness.
  • Challenge: Ensuring navigation remained fixed without overlapping content.
  • Solution: Added padding property to prevent overlap of fixed navigation bar.

Lesson Learned

  • The importance of accessibility, ensuring all content is structured with appropriate heading levels.
  • Improved understanding of flexbox for layout design and positioning.
  • Enhanced knowledge of CSS properties for text effects and background styling.
  • Increased proficiency in using media queries for responsive design.
